üéâ Version: v1.0, Release: 2023/08/14

üéâüéâüéâ Keptab First Milestone

Keptab's code has been completely rewritten for new. Frankly, it is a new extension and no more historical technical liabilities.

  • Supported Chrome Extension Manifest V3, new protocol, quick response.
  • Supported chrome.favicon to fetch website favicons, protect privacy.
  • Added Keptab Data Recovery and Keptab will never lost data.
  • Added fully support for Drag'and'Drop
    • handle Drag'a'Drop placed drop placeholder.
    • smooth Drag'a'Drop user experience.
  • Suppored light/dark mode in all pages.
  • Added Copy links to clipboard to tabs list's menus.
  • Fixed press Enter key when rename tabs group.
  • Added press CTRL/SHIFT/META keys will directly open link.
  • Added Navigation Tree, sticky on page right.
    • Tree will scroll automatically followed the lists.
    • Tree will highlighted automatically followed the lists.
  • Fixed update total tabs when added tabs or deleted tabs.
  • Added 49+ languages translations all over the world. Surprised by AI!

‚ú®‚ú®‚ú® New Feature: Navigation Tree

For the convenience of quick positioning and searching all the named tabs list, we added Navigation Tree that sticky on page right. This would help you locate the item you need quickly.

If you want to make the page more simplicity, you would turn this off on Keptab options page.

‚ú®‚ú®‚ú® New Feature: Keptab Data Recovery

Although Chrome claims that IndexedDB will not be lost as a transactional database, in actual use, there are still frequent instances of data loss, especially when cleaning browser cache, which can easily cause plugin data loss.

Keptab V1.0, we added recover data mode in Keptab Options, you can goto Keptab Options page to recover your data and then re-import Keptab.

‚ú®‚ú®‚ú® New Feature: 49+ Languages Translations

By using AI, we added 49+ languages translations all over the world. Keptab supported almost every language that people used.

‚ú®‚ú®‚ú® New Feature: Fully supported Drag and Drop

Keptab now has fully support for Drag'n'Drop. when you drag tab item, Keptab will placed a simple placeholder clearly.
