About Keptab

Keptab: Kept tabs as tabs group and read them later

Keptab is simple browser tabs manager

Whenever you find yourself with too many tabs, click the Keptab icon to convert all of your tabs into a list. When you need to access the tabs again, you can either restore them individually or all at once.

  • Save open tabs as collections that can be easily restored later. Great for freeing up memory and avoiding clutter.
  • It is designed to be lightweight yet featureful. Keptab based on IndexedDB that can save over 30000 tabs.
  • See and manage all open tabs in one place. With the Keptab instant search, you can search anything you saved.
