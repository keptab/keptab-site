Backup Data

Backup data for Keptab

Standard Backup

Keptab allows for the export of all current tabs group(tabs list). You can do this:

  • click Export Data in the Keptab page right top.
  • Goto Keptab Options and click the Export Data

All current tabs group(tabs list) exported as JSON file. Jsut save this exported file to another place or sync to others services.

Advanced Backup

Just backup the browser file for later recovery.

Find the Current Chrome Profile Folder

A Keptab installation is specific to a single Chrome profile.

To locate Keptab data files, first navigate to the folder corresponding to the currently-active Chrome profile.

Locate the Current Chrome Profile Folder

Chrome allows for the use of multiple user profiles. If you don’t have multiple user profiles set up in Chrome, then your settings and installed extensions are associated with the Default profile.

To locate the folder of the Chrome profile for this window, do the following

  1. Click and drag the text below into the Chrome title bar above to open a new tab containing Chrome version information:

    chrome://version/

  2. Copy the value of the Profile Path displayed on that page.

    // On Windows, it looks something like this:
C:\Users\[YOUR_USERNAME_HERE]\AppData\Local\Google\Chrome\User Data\[YOUR_PROFILE_NAME]
// [YOUR_PROFILE_NAME] is always "Default"

  3. Navigate to the profile path copied in the previous step.

Keptab Data

Once you have navigated to a Chrome profile folder, the Keptab data associated with that profile can be found within the folder as follows:

Keptab lists data is stored in this subfolder:

// please backup all files in this folders
IndexedDB\chrome-extension_feodekcipceogpeomnijgmdlmocbggbi_0.indexeddb.leveldb\

Keptab Settings data is stored in these files:

// please backup all files in this folders
Local Extension Settings\feodekcipceogpeomnijgmdlmocbggbi\

Recovery Backup

Although unlikely, a browser crash or malfunction can lead to the corruption or deletion of data in a way that makes it difficult or impossible for Keptab to automatically recover it.

You can use Keptab Data Recovery in Keptab Options.

This data recovery driven by browser history items, in other words, this data will never lose.
