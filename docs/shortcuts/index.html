Keptab Shortcuts

Keptab has defined (must hard-code) some keyboard shortcuts for user and shortcuts would make things done quickly.

Show Keptab Index

You can use the following keyboard shortcut to display Keptab index page.

  • windows: Alt + K
  • Mac: Option + K

Store All Tabs

You can use the following keyboard shortcut to store all tabs in current browser window. The current window means the window that was most recently focused â€” typically the window on top.

  • windows: Alt + A
  • Mac: Option + A

Store Current Tabs

You can use the following keyboard shortcut to store all current tabs. The current tabs is including the tabs you highlighted or selected tabs (one or more than one tabs) in the currently-focused window (or most recently-focused window, if no Chrome windows are focused). You can use this to store the tabs you highlighted.

  • windows: Alt + C
  • Mac: Option + C

Store All Tabs in All Windows

You can use the following keyboard shortcut to store all tabs in all browser windows.

  • windows: Alt + W
  • Mac: Option + W

How to Change Keptab Shortcuts

Keptab has defined (must hard-code) some keyboard shortcuts for user. But if you don't like or make conflict with others apps, you can modify those shortcuts as follow tips.

Chrome/Edge Browser

End users are free to remap all global keyboard shortcuts to their preferred key combination using the UI exposed at chrome://extensions/shortcuts, yes, every extensions would been rewritten it shortcuts with this.

  • Goto the chrome://extensions/shortcuts URL
  • Click the "Pencil" icon to change the shortcut

Change the shortcut just simple to click the keyboard with the key you like!

Attention please, Chrome/Edge browser MUST have requirements as follow:

  • Extension command shortcuts must include either Ctrl or Alt.
    • Modifiers cannot be used in combination with Media Keys.
  • On macOS Ctrl is automatically converted into Command.
    • To use the Control key on macOS, replace Ctrl with MacCtrl when defining the "mac" shortcut.
    • Using MacCtrl in the combination for another platform will cause a validation error and prevent the extension from being installed.
  • Shift is an optional modifier on all platforms.
  • Search is an optional modifier exclusive to ChromeOS.
  • Certain operating system and Chrome shortcuts (e.g. window management) always take priority over Extension command shortcuts and can not be overwritten.

Key combinations that involve Ctrl + Alt are not permitted in order to avoid conflicts with the AltGr key.

Firefox Browser

You can remap Firefox browser keyboard shortcuts to their preferred key combination as follow:

  • Goto the about:addons URL
  • Click the "dots" icon next to the Keptab extension, then click "Manage"
  • Click the "cog" icon, then click "Manage Extension Shortcuts"
